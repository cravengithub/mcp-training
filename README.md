# MCP Training Projects

Kumpulan proyek MCP Server dari pelatihan.

## Daftar Proyek

| Proyek | Deskripsi | Folder |
|--------|-----------|--------|
| Skeleton Server | MCP Server dasar | `/hari1` |
| Database Server | MCP Server dengan MongoDB | `/hari2-db` |
| API Server | MCP Server dengan API eksternal | `/hari3-api` |
| Security Server | MCP Server dengan validasi keamanan | `/hari3-security` |
| Tools Server | MCP Server dengan berbagai tools | `/hari3-tools` |
| Docker Hello | Latihan Docker | `/docker-hello` |

## Cara Menjalankan

Masuk ke folder proyek, lalu:
```bash
npm install
npm run build
npm start
